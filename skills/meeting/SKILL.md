# Meeting

Create meeting agendas, capture minutes, extract action items, and prepare follow-ups.

## Activation

Use when: user asks for "meeting agenda", "meeting minutes", "action items",
"meeting prep", "follow-up", or shares meeting notes to process.

## Workflow

### Before a Meeting (Agenda)
1. **Clarify**: What's the meeting about? Who's attending? How long?
2. **Build**: Create structured agenda with time allocations
3. **Deliver**: Save as .docx or present inline

### After a Meeting (Minutes)
1. **Ingest**: User pastes or uploads raw notes/transcript
2. **Extract**: Key decisions, action items (who/what/when), open questions
3. **Structure**: Format into professional minutes
4. **Deliver**: Save as document, highlight action items

## Agenda Structure
- Meeting title, date, time, attendees
- Purpose / objective (one sentence)
- Agenda items with time allocations
- "Parking lot" section for off-topic items
- Next steps / action items from previous meeting

## Minutes Structure
- Attendees (present + absent)
- Key Discussion Points (summarised, not transcript)
- **Decisions Made** (highlighted)
- **Action Items** (who, what, by when — in a table)
- Open Questions / Parking Lot
- Next Meeting Date

## After Delivery

- "Want me to email these minutes to the attendees?"
- "Should I create a task list from the action items?"
- "Want me to set up the agenda for the next meeting?"
