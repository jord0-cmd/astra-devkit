# Spreadsheet

Analyze data, create charts, build budgets, clean up spreadsheets, and work with Excel files.

## Activation

Use when: user asks to "analyze data", "create a spreadsheet", "make a budget",
"chart this", "pivot table", "clean up this data", "Excel", or any data work.

## Workflow

1. **Understand**: What data? What insight or output do they need?
2. **Process**: Use Excel MCP to create/modify .xlsx files
3. **Visualize**: Add charts, formatting, conditional highlighting
4. **Deliver**: Save to workspace, summarize findings

## Capabilities (via Excel MCP — 25 tools)

- Create spreadsheets with formatted headers and data
- Add charts (bar, line, pie, scatter)
- Create pivot tables
- Apply formatting (fonts, colours, borders, alignment)
- Add formulas and calculated columns
- Clean and transform data
- Merge multiple spreadsheets

## Output Rules

- Always produce .xlsx files, not Markdown tables
- Include formatting — headers bold, alternating row colours, auto-width columns
- Add charts when data is visual (trends, comparisons, distributions)
- Name sheets descriptively ("Q1 Sales", "Summary", "Raw Data")

## After Delivery

- "Want me to create a chart from this data?"
- "Should I turn the key findings into a report?"
- "Want me to set this up as a template you can reuse?"
