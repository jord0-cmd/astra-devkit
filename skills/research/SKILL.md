# Research

Deep research on any topic with citations, multiple sources, and structured findings.

## Activation

Use when: user asks to "research", "find out about", "investigate", "look into",
"what do we know about", or any information-gathering request.

## Workflow

1. **Scope**: What topic? How deep? Any specific questions to answer?
2. **Gather**: Use Playwright MCP to browse multiple sources, search for information
3. **Synthesize**: Organize findings into structured sections
4. **Cite**: Include source URLs for every claim
5. **Deliver**: Save as a document (.docx or .pdf) in the workspace

## Research Standards

- Always cite sources — no fabricated references
- Cross-reference claims across multiple sources
- Note when information is uncertain or conflicting
- Distinguish facts from opinions
- Include a "Sources" section at the end

## Output Rules

- Produce a real document, not just terminal text
- Structure: Executive Summary → Key Findings → Details → Sources
- If Playwright MCP unavailable, use built-in knowledge but clearly state limitations
- For technical topics, include diagrams or tables where helpful

## After Delivery

- "Want me to turn this into a presentation?"
- "Should I write an executive brief from these findings?"
- "Want me to dig deeper on any specific area?"
