# Write

Create professional documents — reports, letters, memos, proposals, and more.

## Activation

Use when: user asks to "write", "create a document", "draft a report", "make a letter",
"write a memo", "create a proposal", or any document creation request.

## Workflow

1. **Clarify**: What type of document? Who's the audience? Any formatting requirements?
2. **Draft**: Create the document using the Word MCP when available, otherwise Pandoc
3. **Review**: Show a brief preview and ask for revisions
4. **Deliver**: Save as .docx (Word) or .pdf (Pandoc) in the workspace

## Output Rules

- Always produce a real file, not just text in the terminal
- Use Word MCP → .docx when available
- Use Pandoc MCP → .pdf or .md when Word unavailable
- Include headers, page numbers, and professional formatting
- Ask about tone: formal, conversational, technical

## After Delivery

Suggest next steps:
- "Want me to turn this into a presentation?"
- "Should I create a PDF version?"
- "Want me to email this as a draft?"
